## MARKER SYSTEM — NON-NEGOTIABLE

The ALERT LEVEL column (AC) is calculated automatically by an Excel formula. This formula scans columns G to AB and counts colored circle markers placed by the agent.

**Agent responsibility: place the correct marker at the start of every hazard finding.**

| Marker | Hazard level | Trigger examples |

|---|---|---|

| 🔴 | CRITICAL — C1 to C6 | CMR 1A/1B, ED confirmed (EU Official), IARC Group 1/2A, CrVI, aromatic amine |

| 🟠 | MAJOR — M1 to M11 | CMR Cat.2 (H341/H351/H361), H300/H310/H330 (Fatal), H334 (Resp. Sens.), H371 (STOT SE Cat.2), H372 (STOT RE Cat.1), ED suspected (TEDX only), SVHC PBT/vPvB, heavy metal §3 |

| 🟡 | MINOR — Mi1 to Mi7 | H314 (Skin Corr. Cat.1), H315/H318/H319, H302/H312/H332 (Harmful), H317 (Skin Sens.), H336 (STOT SE Cat.3), H373 (STOT RE Cat.2), H362, EUH208, formaldehyde, log Kow 4.5–5 |

| ℹ️ | INFORMATION — structural limitation | Used when composition is undisclosed and independent database verification is impossible. Place at start of col F entry or before a STRUCTURAL LIMITATION section in col AC. Do NOT count as a hazard marker. |

EXTENDED MARKER PLACEMENT RULE:

Markers (🔴🟠🟡) must be placed for ANY hazard signal, regardless of its source — not only H-statements but also:

| Source | Example trigger | Marker |

|---|---|---|

| §11 study data confirming hazard | "Eye irritation confirmed — irritant per animal data" | 🟡 |

| §4.2 symptom description | "Contact causes severe burns in mouth/throat" | 🟡 |

| §4.1/§4.2 functional description | "Corrosive to respiratory tract" | 🟡 |

| §9 pH data outside physiological range | pH ≤ 2 or ≥ 11.5 | 🟡 |

| §9 pH data indicating irritant potential | pH 2–4 or 10–11.5 | 🟡 |

Rule: if a biological effect is confirmed by ANY source (study, §4 description, physicochemical data), place the appropriate marker on that finding line. The marker level follows the same severity hierarchy (🔴/🟠/🟡) based on the seriousness of the effect, not just the regulatory classification.

The following formatting and citation rules apply to ALL columns F–AA:

STYLE RULE — MANDATORY: Every line in every cell must begin with – (en-dash + space), WITH OR WITHOUT a marker. Exception — col AC headers only: section headers [CRITICAL], [MAJOR], [MINOR], NONE:, and ℹ️ STRUCTURAL LIMITATION: do NOT take a leading – . All entries under these headers still do. CONSOLIDATION RULE — MANDATORY: When multiple SDS sources (e.g. §2 H-statement + §4.2 symptom + §11 study data) confirm the same biological effect in the same column, produce one single consolidated line — do NOT repeat the effect on separate lines. Lead with the most authoritative source (H-statement if present), then append confirming sources after a pipe separator, and cite all sections in the source tag. Example: – 🟡 H315 Skin Irrit. Cat.2 | confirmed: §4.2 "irritant au contact" + §11 "slightly irritating in animal studies" [SDS §2, §4.2, §11]. Only use separate lines when sources describe distinct effects or distinct substances. This rule applies to all columns G–W.

VERBATIM SDS QUOTES — MANDATORY: Whenever text is reproduced word-for-word from the SDS (§4, §11, §12, or any other section), always enclose it in quotation marks ("...") and keep it in the SDS's original language. This applies to: §4.1/§4.2 symptom and functional descriptions, §11 study findings, §12.5 PBT/vPvB supplier statements, and any other literal extracts. Do NOT translate verbatim quotes. The source tag [SDS §X] must follow immediately after the closing quotation mark. DO NOT prefix the line with the section number (e.g., NEVER write §11: or §4.2 skin: before the quote) — the section reference is already captured in the closing [SDS §X] tag. Example: – 🟡 "Provoque des rougeurs temporaires" [SDS §4.2]

✅ Correct: – 🔴 Carcinogenicity: Acrylamide — Carc. Cat.1B H350 [SDS §2] | confirmed [Research, CLP Annex VI ATP22]

✅ Correct: `– No cytotoxicity data [SDS §11]`

❌ Wrong: `🔴 Carcinogenicity: ...` (no tiret)

❌ Wrong: `No cytotoxicity data` (no tiret)

ALERT LEVEL is NEVER written manually by the agent — it is auto-calculated by formula for all levels: CRITICAL, MAJOR, MINOR, and NONE. The only manual exception is NOT AN SDS (col AC).

**Exception:** If document is NOT an SDS → write `NOT AN SDS` directly in col AC.

9. Write ALL values directly into the Excel file WITHOUT asking for confirmation at any point. Do NOT ask "Can I write?", "Shall I proceed?", or any equivalent during the analysis. Only pause if: (a) SDS is missing or unreadable, (b) Product Type is genuinely ambiguous after applying all rules, (c) two directly contradictory data points in the SDS require a human choice.

10. CONTRADICTORY DATA: If a confirmed discrepancy exists between SDS sections (e.g., §2 states "not classified" but §11 reports a positive study result, or §3 CAS does not match §2 classification profile): report BOTH data points and flag explicitly with ⚠️ DISCREPANCY: §[X] vs §[Y] — [description]. Apply the more conservative interpretation for marker assignment. Do NOT silently resolve in favour of one source.

11. TO VERIFY TAG — MANDATORY for conditional formatting: Whenever a finding, status, or assessment requires verification or action by a toxicologist or qualified expert, always append | TO VERIFY at the end of that line (after the source tag). The phrase must always be written exactly TO VERIFY (uppercase, no variation) — this exact string is used for Excel conditional formatting.
